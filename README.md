# ANN_Classification
Artificial Neural Network (ANN) based classification model built using PyTorch. This project demonstrates data preprocessing, model building, training, and evaluation for a supervised learning classification task.

🚀 ANN Classification using PyTorch

This project implements an Artificial Neural Network (ANN) for solving a classification problem using PyTorch. It covers the complete machine learning pipeline from data loading to model evaluation.

🔧 Features
Data preprocessing and loading using DataLoader
Custom ANN model using nn.Sequential
Training loop with forward pass and backpropagation
Loss calculation using appropriate classification loss function
Model evaluation and performance tracking
🧠 Model Architecture
Input Layer
Hidden Layers with activation functions (ReLU)
Output Layer for classification
⚙️ Tech Stack
Python 🐍
PyTorch 🔥
NumPy
Pandas (if used)
📊 Workflow
Load and preprocess dataset
Convert data into tensors
Build ANN model
Train model using optimizer and loss function
Evaluate model performance
